## Overview

A comprehensive collection of reusable agent skills designed for LLM users. Skills extend the capabilities of AI agents and assistants—automating workflows, generating documentation, optimizing code, and more. Each skill is fully documented and ready to install.

## Features

- **Production-Ready Skills** — Thoroughly tested and documented, ready to integrate immediately
- **Easy Installation** — Simple setup process with minimal configuration
- **Modular Design** — Use individual skills or combine multiple for powerful workflows
- **Clear Documentation** — Each skill includes usage instructions and examples
- **Community Focused** — Built for and by LLM users solving real-world problems

## Available Skills

### [README ](readme/)
Generate professional, visually polished GitHub `README.md` files for any project. Features polished headers with badges, auto-detection of project structure, and best-practice documentation templates.

**Use when:** Creating project documentation, writing repo READMEs, or generating project overviews.

## Tech Stack

- **Format:** Markdown + YAML frontmatter
- **Documentation:** Standard skill format with usage guidelines
- **Target:** AI agents, GitHub Copilot, and LLM assistants

## Getting Started

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/USERNAME/skills.git
   cd skills
   ```

2. **Install a skill** — Each skill folder contains a `SKILL.md` file with full documentation and implementation details.

3. **Integrate with your agent** — Follow the skill's setup instructions to register it with your AI assistant or LLM platform.

### Using a Skill

Each skill includes:
- **Description** — What the skill does and when to use it
- **Configuration** — Any setup required before use
- **Examples** — Real-world usage patterns

Refer to the individual skill directory's `SKILL.md` for complete details.
